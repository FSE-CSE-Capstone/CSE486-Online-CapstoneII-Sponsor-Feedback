Please follow these steps to evaluate students:

  1. Select the project you are evaluating from the list.
  2. Rate each student by choosing the option that best reflects their performance.
  3. Provide optional comments in the box below to share any additional feedback — this is encouraged but not required.
  4. When finished, click Submit ratings for project. Use Back to return to the previous screen if needed.
A note on scoring: Please don't feel pressure to give high marks across the board. Honest, constructive feedback — including areas for improvement — is what helps students grow and is what we value most. Critical scores will not significantly penalize students' final grades. We trust your professional judgment.

If you prefer to provide general feedback for the team as a whole, complete only the last row of each rating matrix (labeled for "Team Overall") instead of rating each student individually.

  • 1 - Far Below Expectations (Fail) = 50%
  • 2 - Quite a Bit Below Expectations = 60%
  • 3 - Somewhat Below Expectations = 70%
  • 4 - Meets Expectations = 80%
  • 5 - Somewhat Above Expectations = 90%
  • 6 - Quite a Bit Above Expectations = 100%
  • 7 - Exceeds Expectations (A+) = 110%
